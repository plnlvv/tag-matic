# tag-matic
*tag-matic (automatic tagging)* — проект, реализованный в рамках курса майнора ИАД, по автоматизации тематического кодирования качественных социологических интервью с помощью моделей генерации кодов и разметки текста.

**Проект:** Автоматизация тематического кодирования качественных интервью в социологических исследованиях

**Команда:** Львова Полина, Андреева Валентина, Чукавин Максим

**Актуальность**
При работе с качественными социологическими данными, такими как интервью, исследователи сталкиваются с огромным количеством длительной работы, которую необходимо выполнять человеку. В частности, это касается кодирования интервью. Чаще всего перед исследователем стоит задача разметить релевантные поставленной теме части интервью, а также вынести из них коды - ключевые идеи размеченных частей. Некоторые социологи заранее имеют коды из существующих теорий, поэтому им необходима только разметка текста в соответствии с кодами.

**Цель**
Автоматизировать процесс выделения кодов в соответствии с темой исследования и разметки релевантных кодам частей интервью.

**Задачи**

* Генерация кодов (выполняется только в том случае, если у исследователя заранее нет списка кодов)
Вход: текст интервью + тема исследования
Выход: список кодов

* Разметка текста
Вход: текст интервью + список кодов
Выход: размеченный текст в формате <код>цитата</код>

**Данные**
Анонимизированные интервью из лаборатории ВШЭ СПб “Центр молодежных исследований”, коды, соответствующие кодам фрагменты интервью.

**Архитектура для задачи**
* Этап 1 (генерация кодов): дообучение T5 на генерацию списка кодов (sequence-to-sequence).
* Этап 2 (разметка текста): дообучение BERT на multi-label token classification (+ отдельный класс <unk> для обнаружения новых тем).

# tag-matic
tag-matic (automatic tagging) — проект, реализованный в рамках курса майнора ИАД, по автоматизации тематического кодирования качественных социологических интервью с помощью моделей генерации кодов и разметки текста. Разработан командой: Андреева Валентина, Львова Полина, Чукавин Максим — для упрощения работы исследователей ВШЭ СПб.
